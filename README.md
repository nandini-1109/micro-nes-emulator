# Micro NES Emulator (In Progress)

A minimal Nintendo Entertainment System (NES) emulator built from scratch to understand
low-level systems concepts such as CPU emulation, memory mapping, and instruction execution.

This project focuses on **core correctness and system design**, not full game compatibility.
